Hardware Gait Analyzer (Personal Project)

Project Description

This project is a proof-of-concept low-cost gait analysis tool. This prototype uses a pair of wireless sensor modules strategically placed under the L4 vertebra and on either the left or right thigh to measure basic hip flexion and extension movements.

The project provides the possibility for a completely modular and portable gait analyzer, usable in any environment, completely wireless and requiring just a computer to analyse results.

Mechanical Aspect

• Designed and 3D-printed a custom sensor housing for the protoboard holding the microcontroller, circuit wiring, IMU sensor, and LiPo battery.


• Modified the IMU's placement and orientation on the thigh, relying on pitch and roll angles instead of the yaw angle as it's known to drift and provide increasingly wrong values the more measurements are taken.


• Used 2 sensors as the 2nd sensor placed under the L4 vertebra would provide a more accurate reference angle than using a gravity vector normal to the ground when measuring absolute hip flexion and extension.

Electrical Aspect

• Built both sensor modules on protoboards with soldered components.


• Implemented LiPo battery power through an external JST connection, wireless operation within the 3D-printed housing.

Software Aspect

• Collected and processed IMU angle data using Jeff Rowberg's MPU6050 library and the Arduino IDE.


• Developed Python scripts with serial and threading for Bluetooth data logging; currently expanding to WiFi/TCP for simultaneous data collection from both sensors.


• Integrated preliminary gait visualization in OpenSim's biomechanics app using their open-source GUI tools.
