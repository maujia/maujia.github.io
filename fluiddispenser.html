Automated Fluid Dispenser (AFD)

Project Description

A prototype fluid dispenser for use in medical facilities with hands-free dispensing, using an IR sensor, pump, vials, turntable and a custom gearbox. An idea based on Vancouver's pilot project machine called MySafe.

Mechanical Aspect

• Modelled & 3D printed the main structure of the gearbox & fitted it with a 5-gear, 167:1 gear ratio, focusing on high torque for more precise vial rotations and alignment with dispensing nozzle.

Software Aspect

• Integrated all dispenser components using a control loop, running on an Arduino Uno using C code.
