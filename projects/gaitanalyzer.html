Hardware Gait Analyzer

Personal Project

Project Description

A proof-of-concept low-cost motion analysis tool. Using a pair of wireless sensor modules placed under the L4 vertebra and on either the left or right thigh to measure basic hip flexion and extension movements. The project provides the possibility for a completely modular and portable motion analyzer, usable in any environment, completely wireless and requiring just a computer to analyse results.

Key Features

  • Custom wireless sensor modules using ESP32 microcontrollers and MPU6050 inertial measurement sensors
  • 3D-printed sensor housing for protoboard, circuit wiring, and LiPo battery
  • Wireless operation and communication through LiPo batteries and built-in ESP-NOW protocol
  • Data filtered using a Madgwick filter and exponential smoothing
  • Motion analysis performed by looking at the data’s peak angle values and steadiness with Python & SciPy
