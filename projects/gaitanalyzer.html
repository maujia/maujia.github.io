Hardware Gait Analyzer

Personal Project

Project Description

A proof-of-concept low-cost gait analysis tool. Using a pair of wireless sensor modules placed under the L4 vertebra and on either the left or right thigh to measure basic hip flexion and extension movements. The project provides the possibility for a completely modular and portable gait analyzer, usable in any environment, completely wireless and requiring just a computer to analyse results.

Key Features

  • Custom wireless sensor modules using ESP32 microcontrollers and IMU sensors
  • 3D-printed sensor housing for protoboard, circuit wiring, and LiPo battery
  • Wireless operation through LiPo batteries with JST connections
  • Python scripts for Bluetooth data logging with threading
  • Integration with OpenSim's biomechanics application for gait visualization
